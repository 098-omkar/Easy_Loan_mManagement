# Easy_Loan_Management
The platform mainly focus on managing and displaying the data of consumer having active loan accounts.

Steps :

1. open Spring Tool Suite 3.
2. Create Spring starter Project.
3. Add Maven Dependencies.
4. Add 1. Spring JPA, Spring Actuator, Spring Lombok, Spring web, MySql.
5. in src/main/java/ create packages (Layers) such as :
 -> Model -> Repository->Service -> Controller -> main

   For Creating Front END :
   create folders in sub folder such as :
   src/main -> webapps - > views -> jsp pages.
